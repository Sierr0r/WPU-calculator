## ✨ 功能特性

### 实验操作

| 功能 | 说明 |
|:---:|:---|
| **实验配方** | 填写实验信息、添加原料，系统自动计算当量参数 |
| **实验步骤** | 记录合成各步骤的温度、时间、现象和照片 |

<details>
<summary><b>配方设计详情</b></summary>

- 支持 7 种原料类型：多元醇、异氰酸酯、扩链剂(醇类/胺类)、亲水剂、溶剂、其他
- 内置常用原料预设（HDI、MDI、BDO、TEA、EDA 等）
- 自动计算：R 值、HSC（硬段含量）、硬段当量、亲水基团含量
- 支持自定义功能度(f) 和分子量(MW)
- 固含量滑块调节（15%-60%），自动计算理论加水量
- 目标反推：给定目标 R 值或 HSC，反推所需原料质量

</details>

### 数据管理

| 功能 | 说明 |
|:---:|:---|
| **实验历史** | 本地存储所有实验记录，支持多维度查询和排序 |
| **报告可视化** | 从数据库选择实验，实时渲染完整实验报告 |

<details>
<summary><b>数据管理详情</b></summary>

- **本地存储**：所有实验数据存储在浏览器 IndexedDB 中
- **多维度查询**：按编号、日期、操作人、体系搜索
- **灵活排序**：支持按编号、R 值、HSC 等排序
- **数据导入/导出**：
  - 支持 TXT/JSON 格式单实验导入
  - 支持批量历史导入
  - 拖放文件导入
  - 粘贴板导入
- 自动生成配方表、统计图表
- 支持导出 PDF 报告

</details>

### 分析工具

| 功能 | 说明 |
|:---:|:---|
| **成本分析** | 输入原料单价，自动计算配方成本 |
| **参数扫描** | 双变量滑块调节 + 热力图可视化 |

<details>
<summary><b>分析工具详情</b></summary>

**成本分析**
- 原料价格管理：输入各原料单价（元/g）
- 自动计算：原料总成本、固体质量、理论加水量、成品总质量、成品 ¥/kg、固含 ¥/kg
- 价格方案：保存多套价格方案，方便切换对比
- 导入/导出：支持价格表 JSON 格式导入导出

**参数扫描**
- 双变量扫描：选择两种原料，滑块调节质量
- 热力图可视化：探索不同配比下的 R 值与 HSC
- 一键应用：将选定组合应用到当前配方

</details>

### 设置与导出

- **界面设置**：浅色/深色主题、字体缩放（75%-130%）
- **当前实验导出**：复制到剪贴板、导出 TXT、生成 PDF
- **批量处理**：TXT 转 Excel、Excel 批量转 PDF
- **原料预设**：内置常用原料、自定义预设、导出/导入 JSON

### 其他特性

- 多实验标签页同时处理
- 自动保存，防止数据丢失
- 响应式设计，完美适配桌面/平板/手机
- 纯前端实现，离线可用

## [使用说明](#使用说明)

### 基本工作流

```mermaid
graph LR
    A[创建实验] --> B[添加原料]
    B --> C[查看参数]
    C --> D[记录工艺]
    D --> E[保存导出]
```

1. **创建实验**：在实验配方页填写实验基本信息
2. **添加原料**：选择原料预设或手动添加，输入质量
3. **查看结果**：系统自动计算 R 值、HSC 等参数
4. **记录工艺**：在工艺记录页添加实验步骤
5. **保存导出**：生成 PDF 报告或导出数据

### 原料类型说明

| 类型 | 说明 | 常用原料 |
|:---:|:---|:---|
| 多元醇 | 软段主成分 | PPG、PTMG、聚酯二醇 |
| 异氰酸酯 | 硬段主成分 | HDI、MDI、IPDI |
| 扩链剂(醇类) | 醇类扩链 | BDO、HDO |
| 扩链剂(胺类) | 胺类扩链 | EDA、EDA·H₂O |
| 亲水剂 | 引入亲水基团 | DMPA、DMBA |
| 溶剂 | 反应介质 | DMF、NMP、丙酮 |
| 其他 | 助剂等 | 催化剂、消泡剂 |

### 关键参数说明

- **R 值**：NCO/OH 当量比，衡量异氰酸酯过量程度
- **HSC**：硬段含量，影响材料力学性能
- **固含量**：乳液中固体组分质量百分比
